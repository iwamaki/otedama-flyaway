# Otedama Flyaway - ゲーム企画書

## コンセプト
籠から落ちてしまったお手玉が、元の籠に戻るために飛んでいく物理パズルゲーム。

## ストーリー
おばあちゃんの大切なお手玉が、うっかり籠から転がり落ちてしまった。
お手玉は自分の力で飛び跳ねながら、元の籠の中に戻ることを目指す。

---

## 3つの柱

### 1. お手玉の物理挙動
- **重力**: 自然な放物線を描いて落下
- **バウンド**: 地面や壁に当たると跳ね返る（お手玉らしい柔らかさ）
- **回転**: 飛行中にくるくる回転する演出
- **摩擦**: 地面を転がるときの減速

### 2. タッチ操作
- **スワイプで発射**: 引っ張った方向と逆に飛ぶ（パチンコ式）
- **発射の強さ**: スワイプの長さで飛距離が変わる

### 3. ステージ設計
- **ゴール**: 籠（元いた場所）
- **障害物**: 壁、動く障害物、落とし穴
- **ギミック**: 跳ね台、風、転送ポイント
